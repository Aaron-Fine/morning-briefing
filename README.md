# Morning Digest

A self-hosted daily briefing email, generated by Claude, tailored to your interests. Runs as a Docker container on Unraid (or anywhere).

## What It Does

Every morning at 6:00 AM MT, this container:

1. **Collects** data from multiple sources:
   - YouTube Data API (new uploads from your Always Watch channels)
   - RSS feeds (Kagi News, Breaking Defense, SpaceNews, Malcontent News, etc.)
   - Open-Meteo weather API (Cache Valley forecast, no key needed)
   - Yahoo Finance (S&P 500, Dow, oil prices)
   - Come Follow Me lesson schedule (LDS, Old Testament 2026)

2. **Sends** all collected data to Claude via the Anthropic API with a detailed editorial prompt that encodes your preferences (topic priorities, tone, format, etc.)

3. **Renders** Claude's output into a polished HTML email matching your approved format

4. **Delivers** the email to your Gmail via SMTP

## Quick Start

### 1. Get your API keys

| Key | Where | Free? |
|-----|-------|-------|
| Anthropic API | [console.anthropic.com](https://console.anthropic.com/settings/keys) | Pay-per-use (~$1-3/mo) |
| YouTube Data API v3 | [console.cloud.google.com](https://console.cloud.google.com/apis/credentials) | Yes (10,000 units/day) |
| Gmail App Password | [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords) | Yes |

### 2. Configure

```bash
cp .env.example .env
# Edit .env with your API keys

# Edit config.yaml:
# - Set delivery.to_address to your Gmail
# - Adjust topics, schedule, feeds as desired
```

### 3. Test locally

```bash
# Run once immediately
docker compose run --rm morning-digest --now
```

### 4. Deploy to Unraid

**Option A: Docker Compose (if you use the compose plugin)**
```bash
# Copy the project to your Unraid appdata
cp -r morning-digest /mnt/user/appdata/morning-digest
cd /mnt/user/appdata/morning-digest
docker compose up -d
```

**Option B: Unraid Docker template (manual)**
1. In Unraid Docker tab вҶ’ Add Container
2. Repository: build from `/mnt/user/appdata/morning-digest/Dockerfile`
   - Or push to a private registry / GitHub Container Registry
3. Set environment variables:
   - `ANTHROPIC_API_KEY`
   - `YOUTUBE_API_KEY`
   - `SMTP_USER`
   - `SMTP_PASSWORD`
   - `TZ=America/Denver`
4. Mount `/mnt/user/appdata/morning-digest/config.yaml` вҶ’ `/app/config.yaml` (read-only)
5. No ports needed (outbound only)

## Configuration Reference

### config.yaml

**schedule.cron** вҖ” When to send the digest. Default: `0 6 * * *` (6:00 AM daily)

**topics.primary / secondary / tertiary** вҖ” Topic priority tiers. Primary topics always get coverage; secondary when noteworthy; tertiary only when something significant happens.

**digest.at_a_glance.normal_items** вҖ” Target number of headline items (default: 8, max: 12, min: 4)

**digest.quiet_day_policy** вҖ” Set to `always_send` because even one good deep dive justifies the email.

**youtube.always_watch** вҖ” Your 17 channels. Every new upload from these channels appears in the digest. Add/remove channels by editing this list.

**youtube.lookback_hours** вҖ” How far back to check for uploads (default: 48h)

### Adding RSS feeds

Add entries to `rss.feeds` in config.yaml:
```yaml
- { url: "https://example.com/feed.xml", name: "Example" }
```

### FreshRSS integration

When you migrate from Feedly to FreshRSS:
1. Set `rss.provider: "freshrss"` in config.yaml
2. Fill in `freshrss_url`, `freshrss_user`, `freshrss_password`
3. The script will pull unread items from FreshRSS instead of polling individual feeds

### The Atlantic

The Atlantic's website is paywalled and their RSS feed is limited. Options:
- Add their RSS for headlines: `https://www.theatlantic.com/feed/all/`
- For full content, the script would need authenticated access (cookie-based or via their app). This isn't implemented yet вҖ” for now, Claude references The Atlantic's current issue based on what's publicly known.

## Cost

Typical daily cost: **$0.03вҖ“0.08** in Anthropic API usage (one Sonnet call with ~5-10K input tokens, ~3-5K output tokens). Monthly cost should be under $3.

YouTube Data API: Free tier is 10,000 quota units/day. Checking 17 channels uses ~34 units (2 per channel). Well within limits.
